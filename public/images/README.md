# App logo and icons

The admin panel (Filament) uses these automatically (see `AdminPanelProvider`).

| File | Purpose |
|------|--------|
| **light-logo.png** | Logo in sidebar and login (light theme). |
| **dark-logo.png** | Logo for dark mode. |
| **Icon Blue.png** | Favicon for `/admin` (browser tab). |
| **Icon Light.png** | Alternative icon (e.g. light theme). |

After adding or changing files, run: `php artisan view:clear`.
